
  # Astritsi Website

  This is a code bundle for Astritsi Website. The original project is available at https://www.figma.com/design/K9lanWgh2aO7zz5AUQ5AeK/Astritsi-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run 'npm run build', and then push to the github, in order for the website to go live
